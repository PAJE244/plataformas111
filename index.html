📌 Como usar scripts com favoritos

1. Clique em "Copiar Script".

2. Crie um novo favorito no navegador (estrela ⭐).

3. Edite o favorito:

→ Nome: escolha qualquer nome (ex: "Script1")

→ URL: cole o script copiado

4. Salve o favorito.

5. Para usar:

→ Abra o site desejado

→ Clique no favorito OU digite o nome na barra de busca
